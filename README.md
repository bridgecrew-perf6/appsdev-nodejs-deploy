## NodeJs 介绍
- Node.js is a JavaScript runtime built on Chrome's V8
- Node.js uses an event-driven, non-blocking I/O model

### 非阻塞I/O - Non-blocking I/O model
- 阻塞：I/O时进程休眠等待I/O完成后进行下一步
- 非阻塞：I/O时函数立即返回，进程不等待I/O完成
